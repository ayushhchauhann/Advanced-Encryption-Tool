# AES-256 GUI Encryption Application

## Features
- AES-256 encryption and decryption using Python and the `cryptography` library.
- User-friendly graphical interface built with Tkinter.
- Simple operations: Generate Key, Encrypt Files, Decrypt Files.

## How to Run
1. Install dependencies:
   pip install cryptography

2. Run the application:
   python aes256_gui_app.py

## Notes
- A key file `secret.key` will be created. Keep it safe!
- Encrypted files will end with `.enc`, decrypted ones with `.dec`.
